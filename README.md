# Advanced Flight Booking System

A fully functional Java-based flight booking system designed to simulate a real-world airline reservation platform. The application includes secure authentication, role-based dashboards for administrators and customers, dynamic pricing visualization, wallet payments, loyalty rewards, and complete booking management functionality.

## Features

* Secure user registration and login with validation
* Role-based dashboards (Admin and Customer)
* Dynamic pricing with visual indicators
* Wallet system with automatic payment deduction
* Loyalty tier system with discounts
* Booking, cancellation, and persistent data storage
* Comprehensive validation and error handling

## Technical Highlights

* Modular and scalable architecture
* Implementation of core OOP principles: encapsulation, inheritance, polymorphism, abstraction, and composition
* GUI-based interface with real-time updates
* Structured project organization for maintainability
